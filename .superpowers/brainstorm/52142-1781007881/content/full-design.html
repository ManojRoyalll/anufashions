1. Design System (Component Library)

Foundation
  • ✦ Earthy green + terracotta colour tokens
  • ✦ FormField — label + input + error wrapper
  • ✦ Modal — centred overlay with backdrop
  • ✦ DataTable — sortable, searchable, paginated
  • ✦ StatCard — KPI with trend arrow + colour accent
  • ✦ PageHeader — title + action buttons
Shared UI Patterns
  • ✦ All forms: React Hook Form + Zod validation
  • ✦ All modals: open/close state via Zustand
  • ✦ All charts: Recharts (already installed)
  • ✦ Image upload: base64 preview, stored as URL
  • ✦ Toast notifications on save/error

2. Master Data Pages (Categories, Price Ranges, Suppliers)

Categories
  • Name + description
  • Active / inactive toggle
  • Item count badge
  • Edit + delete inline
Price Ranges
  • Name (e.g. "Premium")
  • Min price – Max price
  • Auto-tag items on save
  • Items in range count
Suppliers
  • Name, phone, email
  • Address + notes
  • Outstanding payments
  • Products supplied list
All three use the same layout: table list on the left, + Add button opens a centred modal form.

3. Add / Edit Item Modal

Required Fields
  • 📝 Title / Name
  • 📂 Category (dropdown)
  • 💰 Purchase price
  • 🏷 Selling price
  • 📦 Quantity
  • 🏭 Supplier (dropdown)
Optional Fields
  • 📊 Price range (auto-set + manual override)
  • 🎯 Discount limit % (max allowed discount)
  • 🖼 Images (upload, preview)
  • 🎨 Colour, Size, Material
  • 📋 Notes
  • 🔖 MRP
Live Margin Preview (in modal)
As you type buy/sell prices → shows margin ₹ and % in real time inside the form.

4. Products Page (Stock Management)

Filter bar: Search by name/code · Filter by category · Filter by price range · Filter by stock status · Filter by supplier
Table columns: Code · Name · Category · Price Range · Buy · Sell · Margin · Margin % · Qty · Discount Limit · Status · Actions
Actions per row: Edit (opens modal) · Delete
Summary bar: Total items · Low stock count · Out of stock count · Total inventory value

5. Analytics Dashboard

KPI Cards (with trend ↑↓)
  • Total Investment
  • Total Inventory Value
  • Total Revenue
  • Total Profit
  • Monthly Profit
  • Today's Sales
  • Stock Remaining
  • Net Profit (after expenses)
Charts & Analytics
  • 📈 Daily sales trend (area chart)
  • 📊 Category-wise profit (bar chart)
  • 🥧 Inventory distribution (pie chart)
  • 🏆 Top selling products list
  • ⚡ Investment recovery progress bar
  • 📉 Margin analysis by price range
  • 🔴 Low stock alerts panel

6. Build Order

  1. Design tokens + component library (colours, StatCard, Modal, FormField, DataTable)
  2. Masters pages — Categories, Price Ranges, Suppliers (all similar pattern)
  3. Products page — Add Item modal with all fields, product table with filters
  4. Dashboard rebuild — KPI cards with trends, all charts, alerts panel
  5. Sales & Purchases — polish existing pages, wire discount limit into POS
